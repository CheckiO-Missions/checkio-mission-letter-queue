So, how my final solution should look like?

You just need to join all previous hints and wrap your code into function expression. Using type hints for input and output data are highly recommended.

Here is the whole solution:

def letter_queue(commands: list[str]) -> str:

    queue = []
    for com in commands:
        if com.startswith("PUSH"):
            queue.append(com[-1])
        elif queue:
            del queue[0]
        
    return "".join(queue)

THe following solution is generally the same, but instead of creating and modifying a list you create an empty string directly. Split a command and concatenate a letter to the queue or slicing it from the second element. 

def letter_queue(commands: list[str]) -> str:

    queue = ''
    for com in commands:
        if 'PUSH' in com:
            queue += com.split(' ')[1]
        elif queue:
            queue = queue[1:]
            
    return queue
