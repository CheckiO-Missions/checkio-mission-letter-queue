Proszę złapać pomysł 💡! Proszę zadeklarować pustą listę jako kolejkę. Iterować po poleceniach i za pomocą instrukcji warunkowej wykonać odpowiednie operacje w przypadku konkretnych poleceń: dodać literę jako nowy element listy lub usunąć pierwszy element listy (jeśli lista nie jest pusta). Proszę połączyć pozostałe elementy listy w jeden ciąg i zwrócić go.

Ok, od czego powinienem zacząć?

Proszę zadeklarować zmienną queue z wartością początkową pustej listy. 

queue = []

Iterować po commands używając pętli for. 

for com in commands:

Co robić dalej?

Teraz może Pan mieć dwa warianty poleceń, wciśnij literę lub pop. To dobre miejsce na oświadczenie if/elif. 

if ...:
    ...
elif ...:
    ...
Nadal potrzebuję podpowiedzi.

Opiszmy przypadek z wciśnięciem litery. Takie polecenie zawsze zaczyna się od "PUSH" i kończy pojedynczą literą. Można więc użyć metody str.startswith() z argumentem "PUSH" i dodać ostatni znak polecenia do kolejki za pomocą list.append() i indeksowania ciągów. 

if com.startswith("PUSH"):
    queue.append(com[-1])
Jak opisać drugi przypadek?

Drugi przypadek - "POP" - dotyczy usunięcia pierwszego elementu listy. Proszę sprawdzić, czy lista nie jest pusta, aby dokonać prawidłowego usunięcia i użyć słowa kluczowego del oraz indeksowania listy, aby usunąć pierwszy element. 

elif queue:
    del queue[0]
Jak zwrócić poprawny ciąg znaków?

Proszę użyć metody str.join(), aby otrzymać pojedynczy ciąg z kolejki z pozostałymi literami i zwrócić go. 

return "".join(queue)
Jak więc powinno wyglądać moje ostateczne rozwiązanie?

Wystarczy połączyć wszystkie poprzednie podpowiedzi i zawinąć kod w wyrażenie funkcyjne. Używanie podpowiedzi typu dla danych wejściowych i wyjściowych jest wysoce zalecane.

Poniżej znajduje się całe rozwiązanie:

def letter_queue(commands: list[str]) -> str:

    queue = []
    for com in commands:
        if com.startswith("PUSH"):
            queue.append(com[-1])
        elif queue:
            del queue[0]
        
    return "".join(queue)

Poniższe rozwiązanie jest zasadniczo takie samo, ale zamiast tworzyć i modyfikować listę, proszę utworzyć pusty ciąg bezpośrednio. Rozdzielenie polecenia i konkatenacja litery do kolejki lub wycięcie jej z drugiego elementu. 

def letter_queue(commands: list[str]) -> str:

    queue = ''
    for com in commands:
        if 'PUSH' in com:
            queue += com.split(' ')[1]
        elif queue:
            queue = queue[1:]
            
    return queue
