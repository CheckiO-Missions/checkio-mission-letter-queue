In der Informatik ist eine Warteschlange eine besondere Art von Datentyp, bei dem die Entitäten in der Sammlung geordnet sind und die wichtigsten Operationen mit der Sammlung das Hinzufügen von Entitäten zur hinteren Endposition position (enqueue oder push), und das Entfernen von Entitäten von der vorderen Terminalposition (dequeue oder pop). Dies macht die Warteschlange zu einer First-In-First-Out (FIFO) Datenstruktur. In einer FIFO-Datenstruktur wird das erste Element, das der Warteschlange hinzugefügt wird, auch das erste sein, das entfernt wird. Dies entspricht der Anforderung, dass, sobald ein neues Element hinzugefügt wird, alle zuvor hinzugefügten Elemente entfernt werden müssen entfernt werden müssen, bevor das neue Element entfernt werden kann.

Wir werden den Prozess der Warteschlange mit Python nachbilden. Sie erhalten eine Folge von Befehlen:
- "PUSH X" -- stellt X in die Warteschlange, wobei X ein Buchstabe in Großbuchstaben ist.
- "POP" - die vorderste Position der Warteschlange wird gelöscht. Wenn die Warteschlange leer ist, tun Sie nichts.
Die Warteschlange kann nur Buchstaben enthalten.

Sie sollten alle Befehle abarbeiten und die Buchstaben, die in der Warteschlange verbleiben, in einem Wort vom Anfang bis zum Ende der Warteschlange zusammenstellen hinten in der Warteschlange.

Schauen wir uns ein Beispiel an, hier ist die Reihenfolge der Befehle:
["PUSH A", "POP", "POP", "PUSH Z", "PUSH D", "PUSH O", "POP", "PUSH T"]
